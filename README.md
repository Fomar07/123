# СРС №4: Темная/светлая тема

Учебный проект по теме: переключение светлой и темной темы с сохранением выбора пользователя в `localStorage`.

## Что реализовано

- переключатель светлой и темной темы
- сохранение выбранной темы в `localStorage`
- автоматическая загрузка сохраненной темы при повторном открытии страницы
- аккуратный адаптивный интерфейс на `HTML`, `CSS` и `JavaScript`

## Структура проекта

- `index.html` — основная страница
- `style.css` — оформление светлой и темной темы
- `script.js` — логика переключения и сохранения темы

## Как проверить

1. Откройте `index.html` в браузере.
2. Нажмите на кнопку переключения темы.
3. Обновите страницу.
4. Убедитесь, что выбранная тема сохранилась.

## Технологии

- HTML5
- CSS3
- JavaScript
- localStorage

Как тема применяется на нескольких страницах

Эта страница нужна для демонстрации масштабируемости. Она использует тот же style.css и тот же script.js, поэтому тема ведет себя одинаково во всем проекте.

1. Общий CSS

Обе страницы используют единый набор CSS-переменных для цветов и фона.

2. Общий JavaScript

Один файл управляет переключением темы, чтением localStorage и восстановлением состояния.

3. Общий ключ хранения

Обе страницы читают одно и то же значение preferred-theme в браузере.

Почему это удобно

Для небольшого сайта

  • не нужно дублировать код на каждой странице
  • достаточно подключить общий JS-файл
  • тема будет одинаковой во всех разделах

Для более крупного проекта

  • логику темы можно вынести в отдельный модуль
  • состояние можно хранить локально или на сервере
  • можно добавить дополнительные варианты оформления
